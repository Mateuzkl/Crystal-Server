# Crystal Server

Crystal Server is a free and open-source MMORPG server emulator written in C++.

## Dependencies

### Linux (Ubuntu/Debian)

```bash
sudo apt update && sudo apt install -y \
    git \
    cmake \
    build-essential \
    libluajit-5.1-dev \
    libmysqlclient-dev \
    libboost-system-dev \
    libboost-iostreams-dev \
    libboost-filesystem-dev \
    libboost-locale-dev \
    libpugixml-dev \
    libfmt-dev \
    libspdlog-dev \
    libgmp-dev \
    libcurl4-openssl-dev \
    libprotobuf-dev \
    protobuf-compiler \
    libargon2-dev \
    zlib1g-dev \
    libasio-dev \
    nlohmann-json3-dev \
    libabsl-dev \
    libssl-dev
```

### Windows

Dependencies are automatically managed via **vcpkg**.
